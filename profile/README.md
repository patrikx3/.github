# Powerful Online Networking Tool

Discover the powerful and free online networking tool at [network.corifeus.com](https://network.corifeus.com) and install its MCP server with `npm i -g p3x-network-mcp`.

* **AI Network Assistant**: ask in plain language and get a full domain health report — audit, DNS, SSL, security headers, WHOIS, BGP, DNSBL, IPv6, geolocation, HTTP check, propagation.
* **Mail Tester**: live end-to-end email deliverability — SPF, DKIM, DMARC, spam score and AI-written fix suggestions, results emailed in your language.
* **Monitoring**: TCP, HTTP and Ping checks with alerts, plus customisable public status pages.
* **MCP server**: 17 tools exposed to Claude Code, Codex, Cursor and any MCP client (`claude mcp add p3x-network -- npx p3x-network-mcp`).
